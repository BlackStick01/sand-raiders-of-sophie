# SAND: Raiders of Sophie Wiki

Fan-made wiki site for SAND: Raiders of Sophie.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- next-intl
- MDX content pages

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Verification

```bash
npm run lint
npm run typecheck
npm run build
```

## Production

Primary Vercel deployment:

```txt
https://sand-raiders-of-sophie-wiki.vercel.app
```

Custom domain target:
